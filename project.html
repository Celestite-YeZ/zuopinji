《Everything is Fine》

以伪人、异常与认知错位为核心的都市怪谈叙事游戏。

一个看似正常的世界正在持续产生“异常”。白天，你身处其中却对错误视而不见；夜晚，你成为检查这个世界的人，并在修补漏洞的过程中找到自己的求生之道。
项目类型剧情类微恐 RPG
我的职责世界观设定 / 剧情结构 / 角色设定
展示形式文本设定 / 流程图 / 概念设计

核心概念

项目以“双视角叙事”为核心，玩家将扮演深陷伪人伪造世界的普通人体验日常生活，随后切换为观察员，排查并修复世界中的怪异漏洞，在两个不同视角间切换，逐步发现世界真相。
查看初步剧本

世界观机制

  • 实验体均被植入“认知滤网”系统，使其自动忽略异常，新实验体的滤网需经过 3 天适应期才会固化。
  • 每个观测室都有自毁协议，观察员擅自离岗会被格式化或投入模拟舱成为新实验体。
  • 模拟舱在“实验体出现不可逆变化”时，会启动紧急重置协议，监察室短暂开启封锁。

剧情流程

整体分为 A / B 两个视角世界，进程相互推进。
  • Day 1：实验体 A 启用，正常生活开始；观察员 B 抓住机会解除其认知滤网。
  • Day 2：A 察觉异常并崩溃，B 因看见出逃可能而更加积极介入。
  • Day 3：A 在世界崩塌中逃跑，B 趁机脱逃，伪人展开追逐。

玩法设计

  • 普通人阶段为点击交互，以剧情体验、环境观察和日记记录为主。
  • 观察员阶段根据清单与日记寻找异常点，并选择上报修复。
  • 异常处理以“判断真假异常”为核心，强调信息辨别。

角色设计 · A

  • 身份：被观察实验体 / 学生
  • 背景：新实验体，认知滤网尚未稳定，逐渐在怪异中察觉问题。
  • 习惯：喜欢聊天，记忆力优异，睡前写日记记录见闻。
  • 性格：本性乐观，思维活跃，具备好奇心和质疑勇气。

角色设计 · B

  • 身份：观察员
  • 背景：早年被伪人抓捕，负责检测模拟环境是否符合人类世界。
  • 习惯：长期高压下常自言自语，以此对抗孤独。
  • 性格：深恨伪人，疲惫但坚韧，始终在寻找逃离机会。

设计思考

这个项目的核心不是单纯做“找不同式恐怖”，而是把恐怖感建立在认知系统的崩坏上，并通过 2D / 3D 风格切换突出世界维度不同带来的怪异感。玩法本身因此成为叙事的一部分，而不是独立于剧情之外的小游戏。

Game Jam 内容策划

《风筝凌云志》

限时 48h 命题创作，围绕中国传统风筝元素进行延伸，尝试把收集、成长与图鉴展示组合为完整玩法循环，并加入基础科普信息。

项目类型Game Jam
我的职责内容策划 / 美术资产制作
展示形式试玩视频

设计亮点

  • 通过简单的玩法循环，让玩家在玩的过程中逐步接触不同风筝类型与对应文化内容。
  • 将“放风筝”这一动作完成概念转译，并作为玩法基础。
