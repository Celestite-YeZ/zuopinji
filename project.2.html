《Everything is Fine》

以伪人、异常与认知错位为核心的都市怪谈叙事游戏。

一个看似正常的世界正在持续产生“异常”。白天，你身处其中却对错误视而不见；夜晚，你成为检查这个世界的人，并在修补漏洞的过程中找到自己的求生之道。
项目类型剧情类微恐RPG
我的职责世界观设定 / 剧情结构 / 角色设定
展示形式文本设定 / 流程图 / 概念设计

核心概念

项目以“双视角叙事”为核心，玩家将扮演深陷伪⼈伪造世界的正常⼈，体验⽇常⽣活，随后切换为观察着伪造世界的观察员，排查并修复世界中的怪异漏洞，维持伪造世界的稳定，在两个不同的视⻆间切换，逐步发现世界的真相。

世界观机制

  • 实验体均被植入“认知滤网”系统，使实验体自动忽略异常，但新实验体的“认知滤⽹”并不稳定，需要经过3天的“适应期”后才会固化。
  • 每个观测室都有⾃毁协议，如果观察员擅⾃离开岗位，要么被格式化，要么被投⼊模拟舱成为新实验体。
  • 模拟舱在“实验体出现不可逆变化”时，会启动紧急重置协议⸺监察室的全⽅位封锁将短暂开启。

剧情流程

分为AB两个世界，进程相互推进。
  • Day 1：新实验体A启用，开始一天的正常生活；观察员B抓住机会解除其认知滤网，完成当天任务。
  • Day 2：实验体A察觉世界异常陷入崩溃，观察员B见其觉醒认为出逃有望，即便工作量因此增加也乐此不疲。
  • Day 3：实验体A在世界崩塌中逃跑，观察员B趁机脱逃，伪人对其展开追逐。

玩法设计

  • 普通人阶段为点击交互，以剧情体验、环境观察和日记记录为主，形成观察员视角下的异常线索。
  • 观察员阶段根据清单与日记线索寻找异常点，在“后台”中寻找相应的异常，并选择上报修复。
  • 异常处理以“判断真假异常”为核心，强调信息辨别，失败三次后，当天清单和笔记本上收录的异常刷新。

角色设计

  • 主⻆A：
  • ⾝份：被观察实验体/学⽣
  • 背景：新实验体，认知滤⽹尚未完全稳定，最初跟随“记忆”进⾏⽇常活动，但逐渐在世界的怪异中察觉到问题，开始质疑⾃⾝、质疑世界
  • 习惯：喜欢聊天且记忆⼒优异，每⽇睡前通过写⽇记的⽅式记录⽣活⻅闻
  • 性格：本性乐观，思维活跃，有好奇心和质疑的勇气

角色设计

  • 主⻆B：
  • ⾝份：观察员
  • 背景：早年被伪⼈强⾏抓捕来作为“员⼯”囚禁，负责帮伪⼈检测实验仓的模拟环境是否符合⼈类世界，⼀直渴望能有机会逃出，并⼀直在为此默默筹划
  • 习惯：常年生活在高压生活下，经常絮絮叨叨、⾃⾔⾃语，这是他抵抗经年累⽉的孤独的武器
  • 性格：深恨伪人，疲惫但坚韧，精神紧绷，偶尔会感到绝望，但始终积极思考

设计思考

这个项目的核心不是单纯做“找不同式恐怖”，而是把恐怖感建立在认知系统的崩坏上，并且想通过2D/3D的风格切换来突出世界维度不同的感觉，突出怪异感。玩法本身因此成为叙事的一部分，而不是独立于剧情之外的小游戏。

《风筝凌云志》

限时48h+命题创作，围绕中国传统风筝元素进行延伸，尝试把收集、成长与图鉴展示组合为完整的循环结构，并加入基础科普信息。

项目类型Game Jam
我的职责内容策划 / 美术资产制作
展示形式试玩视频

设计亮点

  • 通过“成长/融合/解锁图鉴”的方式，让玩家在轻玩法中逐步接触不同风筝类型与对应文化内容。
  • 将“放风筝”这一动作完成概念转译，并作为玩法基础。

四月四日殷家宅

一款民俗悬疑推理向恐怖游戏剧本创作，以宅邸调查为线索，通过香料、铜镜、致幻与家族秘密构建层层误导。

项目类型民俗推理 / 剧本创作
我的职责剧情设计 / 人物设定 / 玩法设计
展示形式文字剧本 / 证据链

故事概念

前期以灵异恐怖氛围营造压迫感，后期逐步揭露鬼影、铜镜与致幻的背后都是人为操纵，主题落点在“比鬼神更可怕的是人心”。
